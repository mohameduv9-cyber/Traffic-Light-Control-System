# Traffic Light Control System 🚦

A microcontroller-based Traffic Light Control System designed and simulated for embedded systems applications. This project demonstrates the logic of traffic management using timed sequences and hardware-software integration.

## 📁 Project Structure

The repository contains the following essential files:
* **`code.txt`**: The source code (C/Assembly) used to control the traffic light logic.
* **`traffic light.pdsprj`**: The Proteus Design Suite simulation file.
* **`traffic.hex`**: The compiled binary file ready to be flashed onto the microcontroller.
* **`Report Traffic Light.pdf`**: Detailed documentation and technical report of the project.

## 🛠️ Hardware Components (Simulated)
* **Microcontroller:** ATmega32 A .
* **Display:** Red, Yellow, and Green LEDs for Traffic signaling.
* **Resistors:** For current limiting.
* **Power Source:** 5V DC.

## ⚙️ Features
* **Timed Transitions:** Accurate delay sequences for each light phase.
* **Simulation Ready:** Pre-configured Proteus workspace for immediate testing.
* **Optimized Logic:** Efficient code structure for low-memory footprint.

## 🚀 How to Run
1.  **Simulation:**
    * Open `traffic light.pdsprj` in **Proteus Professional**.
    * Double-click the microcontroller and load the `traffic.hex` file.
    * Press the **Play** button to start the simulation.
2.  **Code:**
    * Review `code.txt` to understand the timing logic and port configurations.
